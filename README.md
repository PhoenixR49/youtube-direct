## Installation
***
```
$ git clone https://github.com/PhoenixR49/youtube-direct.git
$ cd youtube-direct 
$ npm install
$ npm start
```
> Go to http://localhost:3000 to start uploading your videos !

There you go, you have a YouTube Downloader running locally on your computer !
